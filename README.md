# Cedrick Balagtas IS 312 - Area of Circle Calculator
This is a simple Python program to calculate a circle’s area. The user enters the radius, and the program returns the result using Python’s math library.
## How to Run
1. Open the "BalagtasCedrick_AreaOfCircle" folder
2. Open the project "app.py" in PyCharm
3. Run the program
4. Enter the radius when prompted
5. view the calculated are on the console
